# 컬러 믹스: 연금술사 (Color Mix: Alchemist)

> 색을 잃은 세계에 색을 되돌리는 연금술사의 여정 — 색 조합 기반 매치-3 모바일 퍼즐

`Phase 0 - Planning`  `Unity 2D TBD`  `Platform: iOS / Android`  `License: TBD`

## 핵심 컨셉

**컬러 믹스: 연금술사**는 전통적인 매치-3 퍼즐 장르에 **색 조합(Color Mixing)** 메커닉을 결합한 모바일 퍼즐 게임입니다. 플레이어는 세상에서 색이 사라진 회색 세계의 마지막 **연금술사**가 되어, 흩어진 색 블록을 조합하고 연쇄시켜 사라진 명화와 풍경을 하나씩 복원해 나갑니다.

단순히 같은 색 3개를 맞추는 것을 넘어, **빨강 + 파랑 = 보라**, **노랑 + 파랑 = 초록**과 같이 1차색을 조합해 2차·3차색을 만들고, 이를 다시 활용해 더 복잡한 퍼즐을 풀어나가는 **화학/연금술적 깊이**를 제공합니다.

감성적인 내러티브 위에 숫자를 쫓는 랭킹 경쟁과 일일 도전 퍼즐을 얹어, 캐주얼 플레이어와 하드코어 퍼즐러 모두를 만족시키는 것이 목표입니다.

---

## 주요 기능

- **색 조합 시스템** — 1차색(RYB) → 2차색 → 3차색으로 이어지는 조합 트리. 조합 결과에 따라 보드 위의 블록이 화학 반응처럼 변화합니다.
- **연쇄 폭발 (Chain Reaction)** — 하나의 매치가 주변 색을 감염시켜 연쇄적으로 터지는 메타볼/젤리 이펙트. 고연쇄일수록 높은 점수와 특수 블록 획득.
- **프롬프트 / 랭킹 / 배지** — 스테이지별 제시 조건(프롬프트), 주간 리더보드, 특정 조건 달성 시 획득하는 수집형 배지.
- **팔레트 슬롯** — 플레이어가 수집한 색을 보관하고 스테이지 내에서 소환하는 인벤토리형 슬롯. 전략적 자원 관리 요소.
- **갤러리 복원** — 스테이지 클리어 시 명화/풍경의 일부가 색상과 함께 복원되며, 월드맵의 갤러리에서 감상 및 메타 진행 확인 가능.
- **특수 블록** — 프리즘 블록, 필터 벽, 회색 블록(색 도둑), 감염 블록 등 다양한 장애물·기믹.
