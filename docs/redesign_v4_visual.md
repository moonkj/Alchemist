## A. 현 인상 진단 (As-Is)

### A-1. 배경이 플랫 블랙 → "개발자 프로토타입" 느낌
- 현: `#0F1218` 단색 배경 + 패널 `#1A1E29` 단색.
- 문제: L=0.08 근방 단일 톤 → **어둠 속 네온사인** 분위기. SF/호러 톤으로 읽힘.
- 아이들은 "밝은 저녁 하늘 속 마법" 같은 **깊이 있는 색 층**을 기대.

### A-2. 중간 채도 파스텔 실패
- 현: 블록만 채도 높고 배경이 먹색 → 대비가 오히려 **긴장감**(경고등 느낌)을 유발.
- Candy Crush 는 배경이 **풍경화 수채** 톤, Toon Blast 는 **만화 그림책** 톤. 우리는 **톤이 없다**.

### A-3. 이모지 7종만 — 과소
- 현존: 🏆 ⏱ 🎨 🏠 ❓ ⚙ ⏸ (7종, 전부 기능 아이콘).
- 문제: 감정 이모지·테마 이모지·비유 이모지(음식/자연) 부재. 화면이 **정보판**에 가깝고 **놀이터**가 아님.

### A-4. 마스코트 / 캐릭터 0개
- 연금술사 꼬마, 붓 요정, 팔레트 친구 등 **감정 이입 대상 전무**.
- 성공/실패 모두 숫자·별점만으로 반응 → "칭찬하는 얼굴"이 없다.

### A-5. 반짝임(Sparkle) 레이어 부재
- 스테이지 클리어·콤보·매치 때도 파티클은 있으나 **UI 자체**(타이틀, 패널 가장자리, 아이콘 주변)는 정적.
- 게이밍 UI 는 3가지 레이어가 필수: **① 구조 ② 정보 ③ 반짝임**. 우리는 ③이 비어있음.

## B. 톤 업그레이드 — Mood Board 언어

### B-1. Primary Tone: "저녁 노을 → 반짝이는 마법 색연필"

## E. 무브먼트 & 폴리시 이미지

### E-1. 상시 미세 모션 (Ambient)
- **로비 타이틀 "안녕 친구!"**: ±2° 천천히 좌우 흔들림, 주기 3.5초, ease-in-out. 손편지 느낌.
- **스테이지 카드 hover/focus**: 마우스/포커스 없는 모바일에서는 "다음 도전 카드" 1개만 ±1° 느린 흔들림 (주기 2.8초) — 다음 할 일 유도.
- **스파클 파티클**: 타이틀·결과·콤보 주변 상시 3~6개, 랜덤 수직 부유.

### E-2. 탭 피드백 (Tap)
- **스테이지 카드 탭**: pop scale 1.0 → 1.1 → 1.0 (total 250ms, ease-out-back).
- **CTA 버튼 탭**: scale 0.95 → 1.05 → 1.0 (180ms) + 주변 ✨ 4개 방사형 뿜기 (반경 40px, 400ms 페이드).
- **설정 토글**: 슬라이드 180ms + 끝 ✨ 플래시.

### E-3. 전환 (Transition)
- **게임 → 결과**: 보드가 0.4초 동안 페이드+blur 1px, 동시에 "갤러리 미리보기 썸네일" 컷이 우측에서 슬라이드인. 클리어한 그림이 **즉시 보인다** = 성취감 극대화.
- **결과 → 로비**: 화면 위에서 스파클 샤워 0.8초 → 페이드.
- **로비 → 게임**: 선택한 스테이지 카드가 중앙으로 확대 → 화이트 플래시 → 게임 씬.

### E-4. 저사양 모드 대응 (Perf 협의)
- 설정 "화려함 낮춤" ON 시:
  - 그라디언트 → 단색 (기존 톤 유지)
  - 스파클 파티클 0개
  - 레인보우 hue lerp → 고정색 (`#FFD93D`)
  - 타이틀 흔들림 → OFF

## F. 프로그래매틱 구현 힌트 (Coder Teammate 참고)

### F-1. 그라디언트 배경 (OnGUI)
- 옵션 A: `Texture2D` 동적 생성 — 세로 N행(64~128), 각 행을 `Color.Lerp(top, bottom, t)` 로 채워 1회 캐싱. `GUI.DrawTexture(rect, tex)` 로 그리기.
- 옵션 B: 두 단색 `Texture2D` 를 위/아래로 겹치고 중간에 alpha blend 한 세 번째 레이어 — 연산 최소, 품질 타협.
- 권장: A (캐시 1회면 runtime cost 무시 가능).

### F-2. 스파클 (✨ glyph 파티클)
- 기존 `SpawnPaintSplash` 구조 재활용 — paint splash 에서 glyph 만 "✨" (혹은 "·") 로 교체, size 8~14pt, 수명 0.8~1.2초, 중력 -50 (위로 떠오름).
- 별도 메서드 `SpawnSparkle(Vector2 center, int count, Color tint)` 권장.

### F-3. 레인보우 글자
- "STAGE CLEAR" 글자별 색:
  ```
  float hue = (Time.time * 0.3f + charIndex * 0.15f) % 1f;
  Color c = Color.HSVToRGB(hue, 0.7f, 1f);
  ```
- 글자별 GUIStyle 복사 후 `normal.textColor = c` 설정 → `GUI.Label` 로 하나씩 그리기.

### F-4. 이모지 폰트 검증
- Unity 에서 기본 폰트가 컬러 이모지 지원 여부 플랫폼별로 다름.
- iOS/Android 는 시스템 이모지 fallback 있음. Editor 에서는 monochrome 으로 보일 수 있음 — **실기기 QA 필수**.

### F-5. 타이틀 흔들림
- `Time.time` 기반 `Mathf.Sin` 으로 회전각 계산:
  ```
  float angle = Mathf.Sin(Time.time * (2*Mathf.PI / 3.5f)) * 2f;
  ```
- OnGUI 에서는 `GUIUtility.RotateAroundPivot(angle, pivot)` 으로 회전.
